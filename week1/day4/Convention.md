## 📘 Word: **Convention**

---

## 1. 📌 Basic Meaning

A **convention** is a **common rule, agreement, or way of doing something** that most people follow.
(Conventional tareeqa ya rule jo sab log maan lete hain)

---

## 2. 🟢 Roman Urdu Meaning

**Convention = riwayat / mamooli qanoon / agreed rule**
(Jo sab log mil kar follow karte hain ya accept kar lete hain)

---

## 3. ⭐ Why This Word Matters

We use conventions so people can **understand each other easily**.
Without conventions, everyone would do things in a different way.
(Conventions is liye zaroori hain taake sab log same way samajh saken)

---

## 4. 🏠 Daily-Life Usage

* In writing, we follow spelling conventions. (Likhnay mein hum spelling ke rules follow karte hain)
* In school, there are uniform conventions. (School mein uniform ke rules hote hain)
* People follow social conventions when meeting others. (Log miltay waqt social rules follow karte hain)

---

## 5. 🎓 Academic Usage

* Academic writing follows strict conventions. (Academic likhai mein strict rules follow hote hain)
* The research paper uses citation conventions. (Research paper mein citation ke rules use hote hain)
* Students must learn grammar conventions. (Students ko grammar ke rules seekhne hote hain)

---

## 6. 💻 Computer Science Usage

* Programming languages follow naming conventions. (Programming languages naming rules follow karte hain)
* Developers use coding conventions for clean code. (Developers clean code ke liye coding rules use karte hain)
* APIs follow communication conventions. (APIs communication ke fixed rules follow karte hain)
* JSON has formatting conventions. (JSON ke apne formatting rules hote hain)
* Git commits follow message conventions. (Git commits ke message likhne ke rules hote hain)

---

## 7. 🧠 Beginner Explanation

Think of a **convention like a “common agreement”**.
It is not always a law, but everyone follows it so things stay organized.
(Ye ek aisa agreed tareeqa hota hai jo sab follow karte hain taake confusion na ho)

---

## 8. 🔁 Synonyms vs Similar Words

**Synonyms:**

* Rule
* Standard
* Tradition
* Practice

**Similar Words:**

* Law (strict rule by government)
* Guideline (soft instruction)
* Protocol (formal procedure)

---

## 9. ❌ Antonyms

* Chaos (be-qaidgi)
* Disorder (bikhra hua system)
* Randomness (bina rule ke)

---

## 10. 📚 Word Family

* Convention (noun)
* Conventional (adjective)
* Conventionally (adverb)

---

## 11. 💬 Common Phrases

* follow conventions
* social conventions
* coding conventions
* industry conventions
* established conventions

---

## 12. 🧠 Memory Technique

Think: **“CONVENTION = CONVENIENT way everyone agrees on”**
(Convenient tareeqa jo sab log mil kar accept kar lete hain)

---

## 13. ⚠️ Common Mistakes Students Make

* Thinking convention means a **law** (but it is usually just a common rule)
* Confusing it with **invention** (different word!)
* Using it without context (always say *what type of convention*)
* Mixing up formal and informal conventions
