# 📘 Word: **Correctness**

---

## 1. 📌 Basic Meaning

**Correctness** means being **right, accurate, or without mistakes**.
(yaani koi cheez sahi ho aur us mein ghalti na ho)

---

## 2. 🟢 Roman Urdu Meaning

**Correctness = sahi hona / durust hona / ghalti se free hona**
(yaani jab koi answer, code ya idea bilkul theek ho)

---

## 3. ⭐ Why This Word Matters

In studies and programming, **correctness is very important** because:

* Wrong answers can cause failure
* Wrong code can break programs
* Correct results build trust

(agar cheez sahi na ho to system ya result kaam nahi karega)

---

## 4. 🏠 Daily-Life Usage

* Checking homework answers
* Solving math problems
* Giving correct information

---

## 5. 🎓 Academic Usage

* Checking answers in exams
* Verifying scientific results
* Ensuring logical reasoning is right

---

## 6. 💻 Computer Science Usage

In Computer Science, correctness means:

* Code gives the **right output**
* Algorithm works as expected
* No logical errors exist

---

## 7. 🧠 Beginner Explanation

Think of correctness like a **math answer key**.
If your answer matches the key, it is correct.
If not, it is incorrect.
(yaani sahi jawab woh hota hai jo expected result se match kare)

---

## 8. 🔁 Synonyms vs Similar Words

**Synonyms:**

* Accuracy
* Precision
* Validity

**Similar Words:**

* Truth
* Rightness

---

## 9. ❌ Antonyms

* Incorrectness
* Error
* Mistake
* Inaccuracy

---

## 10. 📚 Word Family

* Correct (adjective)
* Correctly (adverb)
* Correctness (noun)
* Incorrect (opposite adjective)

---

## 11. 💬 Common Phrases

* Check correctness
* Ensure correctness
* Prove correctness
* Verify correctness

---

## 12. 🧠 Memory Technique

Think:
👉 **Correctness = “Correct + ness” = state of being correct**
(yaani jab cheez puri tarah sahi ho)

---

## 13. ⚠️ Common Mistakes Students Make

* Confusing correctness with speed
* Thinking “almost right” means correct
* Ignoring small errors in code
* Not testing final output properly

# ✍️ Sentences

---

## 🏠 1. Daily-Life Sentences

* I checked the correctness of my homework answers. (Main ne apne homework ke jawabat ki sahi hona check kiya)
* The teacher praised the correctness of my solution. (Ustaad ne mere solution ki durusti ko tareef ki)
* We should ensure the correctness of information before sharing. (Humein share karne se pehle maloomat ki sahi hona confirm karna chahiye)

---

## 🎓 2. Academic Sentences

* The correctness of the experiment results is very important. (Experiment ke natayij ki durusti bohat zaroori hoti hai)
* Students must focus on the correctness of their reasoning. (Students ko apni soch ki sahi hona par focus karna chahiye)
* The correctness of formulas is checked in exams. (Exams mein formulas ki durusti check ki jati hai)

---

## 💻 3. Computer Science Sentences (ACTIVE VOICE)

* The programmer tests the correctness of the algorithm. (Programmer algorithm ki sahi hona test karta hai)
* The system checks correctness before saving data. (System save karne se pehle data ki durusti check karta hai)
* Developers verify correctness using test cases. (Developers test cases se correctness verify karte hain)
* The compiler ensures correctness of syntax. (Compiler syntax ki sahi hona ensure karta hai)
* The team improves correctness of the software logic. (Team software logic ki durusti improve karti hai)

---

## 💻 4. Computer Science Sentences (PASSIVE VOICE)

* The correctness of the program is checked by the system. (Program ki durusti system ke zariye check ki jati hai)
* Errors are found when correctness is tested. (Errors tab milte hain jab correctness test ki jati hai)
* The algorithm is validated for correctness by engineers. (Algorithm ki sahi hona engineers ke zariye validate ki jati hai)
* The output is verified for correctness by the compiler. (Output ki durusti compiler ke zariye verify ki jati hai)
* The data is checked for correctness before processing. (Data ko process karne se pehle uski sahi hona check ki jati hai)
