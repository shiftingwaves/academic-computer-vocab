# 📘 Word: **prior**

## 1. Basic Meaning

**Prior** means:

* happening **before**
* existing **earlier**
* more important first

(Roman Urdu: pehle se maujood ya pehle hone wali cheez)

---

## 2. Roman Urdu Meaning

* **Prior = pehle**
* **Prior knowledge = pehle se maloomat**
* **Prior experience = pehle ka tajurba**

---

## 3. Why This Word Matters

This word is very common in:

* university books
* research papers
* programming documentation
* exams
* interviews

You will often read:

* “prior knowledge”
* “prior experience”
* “prior to installation”

(Roman Urdu: yeh lafz studies aur CS field mein bohat zyada use hota hai)

---

## 4. Daily-Life Usage

We use **prior** when talking about something earlier.

Example:

* prior meeting
* prior plan
* prior notice

(Roman Urdu: yani koi kaam jo pehle se decide ya ho chuka ho)

---

## 5. Academic Usage

Teachers and books use **prior** in formal English.

Example:

* prior research
* prior learning
* prior information

(Roman Urdu: academic writing mein pehle wali maloomat ke liye use hota hai)

---

## 6. Computer Science Usage

In Computer Science, **prior** is used for:

* prior data
* prior knowledge
* prior version
* prior permission
* prior probability (AI/ML)

(Roman Urdu: CS mein pehle wali information ya version ko batane ke liye use hota hai)

---

## 7. Beginner Explanation

If something happens **before another thing**, we can call it **prior**.

Simple idea:

* prior = before

Example:

* Prior version = old version
* Prior knowledge = knowledge you already had

(Roman Urdu: jo cheez pehle ho ya pehle se ho usay prior kehte hain)

# 11. Common Phrases

* **prior knowledge** → pehle se knowledge
* **prior experience** → pehle ka tajurba
* **prior approval** → pehle se permission
* **prior notice** → pehle se batana
* **prior version** → purana version

# 12. Memory Technique

Think:

👉 **Prior = Pre + Earlier**

Both mean something happens first.

Easy trick:

* “Prior” sounds professional way of saying “before”.

(Roman Urdu: jab bhi “prior” dekho to dimagh mein “pehle” lao)

# 13. Common Mistakes Students Make

❌ Wrong:

* “I prior completed the task.”

✅ Correct:

* “I completed the task prior to submission.”

---

❌ Wrong:

* “Prior than”

✅ Correct:

* “Prior to”
