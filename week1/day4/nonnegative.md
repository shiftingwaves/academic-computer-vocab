# **Word: nonnegative**

## 1. Basic Meaning

**Nonnegative** means:

* **Not negative**
* A number that is **zero or greater than zero**

(Ya number zero ya zero se bara hota hai)

---

## 2. Roman Urdu Meaning

**Nonnegative = Jo negative na ho**

Example:

* 0, 5, 10 are nonnegative numbers.
  (Ye sab zero ya zero se baray numbers hain)

---

## 3. Why This Word Matters

This word is important because:

* It is used in **Math**
* It is used in **Programming**
* It is used in **Algorithms**
* Many computer systems only allow nonnegative values

(Computer science mein bohat jagah negative values allow nahi hotin)

---

## 4. Daily-Life Usage

We use it when talking about:

* Money
* Age
* Scores
* Quantity

Example:

* Your bank balance should be nonnegative.
  (Matlab balance minus mein na ho)

In studies, teachers use this word in:

## 6. Computer Science Usage

In CS, nonnegative values are common in:

* Array indexes
* Loop counters
* Data sizes
* Graph algorithms
* Memory allocation

Example:

* Array index cannot be negative.
  (Is liye index nonnegative hota hai)

---

## 7. Beginner Explanation

Think like this:

* Negative numbers: -1, -5, -10
* Nonnegative numbers: 0, 1, 2, 3

So:
✅ 0 is nonnegative
✅ 9 is nonnegative
❌ -3 is not nonnegative

---

## 8. Synonyms vs Similar Words

| Word            | Meaning                   |
| --------------- | ------------------------- |
| **Positive**    | Greater than zero only    |
| **Nonnegative** | Zero or greater           |
| **Unsigned**    | No negative value allowed |

Important:

* **0 is nonnegative**
* But **0 is not positive**

(Ye beginner students aksar confuse karte hain)

---

## 9. Antonyms

* Negative
* Less than zero

Example:

* -7 is a negative number.
  (Yani zero se chhota number)

## 12. Memory Technique

Break the word:

* **non** = not
* **negative** = minus value

So:

**nonnegative = not negative**

(Simple yaad rakhne ka tareeqa hai)

---

## 13. Common Mistakes Students Make

### ❌ Mistake 1:

Thinking nonnegative means only positive

Wrong:

* “0 is not nonnegative”

Correct:

* “0 is nonnegative”

---

### ❌ Mistake 2:

Using it for all data types

Example:

* Some systems only accept nonnegative integers.
  (Har jagah decimal ya float allow nahi hota)
