## 📘 Word: **Legitimate**

---

## 1. 🎯 Basic Meaning

**Legitimate** means something that is **real, legal, correct, or accepted by rules or law**.
(legitimate ka matlab hota hai koi cheez jo legal, sahi aur rules ke mutabiq ho)

---

## 2. 🗣️ Roman Urdu Meaning

**Legitimate = jaiz, qanooni, ya sahi tareeqay se manzoor shuda**
(yeh wo cheez hoti hai jo rules aur law ke mutabiq ho)

---

## 3. ⭐ Why This Word Matters

This word is important because we use it to check if something is:

* real or fake
* legal or illegal
* correct or not accepted
  (hum dekhte hain ke koi cheez sahi aur qanooni hai ya nahi)

---

## 4. 🏠 Daily-Life Usage

We use this word when talking about rules, jobs, or trust.

Example:

* Is this shop legitimate? (Kya yeh dukan asal aur jaiz hai?)
* He gave a legitimate reason for being late. (Us ne late aane ki sahi wajah di)
* This is a legitimate ticket. (Yeh asal ticket hai)

---

## 5. 🎓 Academic Usage

* This is a legitimate research source. (Yeh aik sahi research source hai)
* The student raised a legitimate question. (Student ne aik valid sawal poocha)
* The argument is legitimate in academic writing. (Taleemi writing mein yeh argument theek hai)

---

## 6. 💻 Computer Science Usage

* The system checks for **legitimate users** before login. (System login se pehle sahi users check karta hai)
* Only legitimate requests are processed by the server. (Server sirf sahi requests process karta hai)
* A legitimate API key is required to access the service. (Service ke liye valid API key chahiye hoti hai)
* The firewall blocks non-legitimate traffic. (Firewall ghalat traffic block karta hai)
* Developers ensure legitimate data input in forms. (Developers forms mein sahi data ensure karte hain)

---

## 7. 🧠 Beginner Explanation

Think of “legitimate” like a **stamp of approval**.
If something is legitimate, it means it is:

* real
* allowed
* trusted
  (agar koi cheez legitimate ho to matlab woh asli aur accept ki gayi hai)

---

## 8. 🔄 Synonyms vs Similar Words

**Synonyms (same meaning):**

* valid
* legal
* genuine
* lawful
* authentic

**Similar words (not exactly same):**

* true (truth focus)
* correct (accuracy focus)
* official (authority focus)

---

## 9. ❌ Antonyms

* illegal
* fake
* fake
* invalid
* unofficial

---

## 10. 📚 Word Family

* legitimate (adjective)
* legitimacy (noun)
* legitimize (verb)
* legitimization (noun form)

---

## 11. 💬 Common Phrases

* legitimate reason
* legitimate concern
* legitimate business
* legitimate user
* legitimate source

---

## 12. 🧠 Memory Technique

Think: **“LEGIT = Legal + Right”**
So legitimate means something that is **legally right and accepted**.
(legit = legal + right → jo sahi aur qanooni ho)

---

## 13. ⚠️ Common Mistakes Students Make

* Using “legit” in formal writing without knowing context
* Confusing “legitimate” with “logical”
* Thinking it only means “true” (but it also means legal/allowed)
* Overusing it in casual speech incorrectly

# ✍️ SENTENCES

---

## 🏠 1. Daily-Life Sentences

* This is a legitimate business in our city. (Yeh hamare shehar ka asli aur jaiz business hai)
* She gave a legitimate excuse for missing class. (Us ne class miss karne ki sahi wajah di)
* I want a legitimate phone, not a fake one. (Mujhe asli phone chahiye, nakli nahi)

---

## 🎓 2. Academic Sentences

* The researcher used only legitimate sources in the report. (Researcher ne report mein sirf sahi sources use kiye)
* The student made a legitimate argument in the essay. (Student ne essay mein valid argument diya)
* This is a legitimate method for solving the problem. (Yeh problem solve karne ka sahi tareeqa hai)

---

## 💻 3. Computer Science Sentences (ACTIVE VOICE)

* The system verifies legitimate users during login. (System login ke waqt sahi users verify karta hai)
* Developers validate legitimate input in the application. (Developers application mein sahi input check karte hain)
* The server accepts only legitimate requests from clients. (Server sirf sahi requests accept karta hai)
* The program detects legitimate transactions in the database. (Program database mein sahi transactions detect karta hai)
* The firewall blocks illegitimate access and allows legitimate traffic. (Firewall ghalat access block aur sahi traffic allow karta hai)

---

## 🖥️ 4. Computer Science Sentences (PASSIVE VOICE)

* Legitimate users are verified by the system before access. (Sahi users ko system access se pehle verify kiya jata hai)
* Valid input data is accepted by the application. (Sahi input data application mein accept kiya jata hai)
* Only legitimate requests are processed by the server. (Sirf sahi requests server ke zariye process ki jati hain)
* Secure transactions are marked as legitimate by the system. (Secure transactions ko system sahi mark karta hai)
* Unauthorized access is blocked and legitimate access is allowed. (Ghalat access block hota hai aur sahi access allow hota hai)
