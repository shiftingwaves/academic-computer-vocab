# 📘 Word: **offset**

## 1. Basic Meaning

**Offset** means:

* to balance something
* to move something a little from its position
* a difference or distance between two things

(Offset ka matlab hota hai kisi cheez ko balance karna ya thora side par rakhna)

---

## 2. Roman Urdu Meaning

* Balance karna
* Farq ya distance
* Thora side par shift karna

---

## 3. Why This Word Matters

This word is very common in:

* programming
* databases
* graphics
* memory management
* daily English

(Ye word Computer Science aur rozana English dono mein bohat use hota hai)

---

## 4. Daily-Life Usage

If your chair is not in the center, it has an **offset**.

(Agar kursi center mein nahi hai to uska position offset hai)

---

## 5. Academic Usage

Teachers use “offset” in:

* mathematics
* physics
* engineering
* computer science

(Teachers ye word technical subjects mein use karte hain)

---

## 6. Computer Science Usage

In CS, **offset** often means:

* a position number
* memory distance
* data location
* starting point

(CS mein offset aksar memory ya data ki position batata hai)

Example:
A program may read data from an offset of 20 bytes.

(Yani program 20 bytes baad wali position se data read karega)

---

## 12. Memory Technique

Think:

👉 “OFF + SET”

Something is **set off** from the normal position.

(Yani cheez apni asal jagah se thori hat gayi hai)

---

## 13. Common Mistakes Students Make

❌ Using “offset” only for movement
✔ It can also mean balance or distance

❌ Confusing offset with “delete”
✔ Offset only changes position or reference

❌ Saying “offset means error”
✔ Offset is not always an error
