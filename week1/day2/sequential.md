## 📘 Word: **sequential**

---

## 1. 📌 Basic Meaning

**Sequential** means things happening in a **specific order**, one after another.

---

## 2. 🧠 Roman Urdu Meaning

**(tarteeb se / aik ke baad aik)**

---

## 3. ⭐ Why This Word Matters

This word is important because many tasks in life and computer science follow steps in order.
If the order changes, the result can become wrong.

---

## 4. 🏠 Daily-Life Usage

We use **sequential** when talking about steps, routines, or instructions done one by one.

---

## 5. 🎓 Academic Usage

Teachers use **sequential** for processes, experiments, or lessons that follow a fixed order.

---

## 6. 💻 Computer Science Usage

In computer science, programs and algorithms often run in a sequential way, step by step.

---

## 7. 🧑‍🏫 Beginner Explanation

If something is **sequential**, it means:

* First step
* Then second step
* Then third step

Everything follows an order.

---

## 8. 🔁 Synonyms vs Similar Words

**Synonyms:**

* ordered
* step-by-step
* consecutive
* successive

**Similar but different:**

* random → no order
* parallel → happening at the same time

---

## 9. ❌ Antonyms

* random
* unordered
* parallel
* mixed

---

## 10. 📚 Word Family

* sequence (noun)
* sequential (adjective)
* sequentially (adverb)

---

## 11. 💬 Common Phrases

* sequential order
* sequential process
* sequential steps
* sequential execution
* sequential data

---

## 12. 🧠 Memory Technique

Think of:
**“Sequence = Steps in order”**
So **sequential** means “following a sequence.”

---

## 13. ⚠️ Common Mistakes Students Make

* Confusing **sequential** with **simultaneous**
* Forgetting that order is important
* Using it for random actions

# ✍️ 1. Daily-Life Sentences (3)

* I followed the recipe in a sequential way. (Maine recipe ko tarteeb se follow kiya)
* The students entered the classroom in sequential order. (Talaba classroom mein aik ke baad aik aaye)
* She arranged the books sequentially on the shelf. (Us ne kitaabein shelf par tarteeb se rakheen)

---

# 🎓 2. Academic Sentences (3)

* The experiment follows a sequential process. (Experiment aik tarteebi process ko follow karta hai)
* The teacher explained the chapters in sequential order. (Teacher ne chapters ko tarteeb se samjhaya)
* The research paper presents ideas sequentially. (Research paper ideas ko aik ke baad aik pesh karta hai)

---

# 💻 3. Computer Science Sentences (ACTIVE VOICE) (5)

* The program executes the instructions in a sequential manner. (Program instructions ko tarteeb se execute karta hai)
* The algorithm processes the data sequentially. (Algorithm data ko aik ke baad aik process karta hai)
* The developer writes the code in sequential steps. (Developer code ko tarteebi steps mein likhta hai)
* The CPU reads memory addresses sequentially. (CPU memory addresses ko tarteeb se read karta hai)
* The system stores records in sequential order. (System records ko aik ke baad aik store karta hai)

---

# 💻 4. Computer Science Sentences (PASSIVE VOICE) (5)

* The instructions are executed sequentially by the processor. (Instructions processor ke zariye tarteeb se execute ki jati hain)
* The files are arranged in sequential order by the system. (Files system ke zariye tarteeb se arrange ki jati hain)
* The input data is processed sequentially by the algorithm. (Input data algorithm ke zariye aik ke baad aik process kiya jata hai)
* The program steps are followed sequentially during execution. (Program ke steps execution ke dauran tarteeb se follow kiye jate hain)
* The database records are stored sequentially on the server. (Database records server par tarteeb se store kiye jate hain)
