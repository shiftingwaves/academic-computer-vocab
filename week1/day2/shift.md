# Word: **shift**

## 1. Basic Meaning

**Shift** means to **move**, **change position**, or **change from one thing to another**.
(Shift ka matlab hai move karna ya position/change badalna)

---

## 2. Roman Urdu Meaning

**Shift = jagah badalna / move karna**

---

## 3. Why This Word Matters

This word is very common in:

* Daily life
* School and university
* Programming
* Computer systems
* Office work

Computer Science students often see:

* **Shift key**
* **Bit shift**
* **Shift operation**
* **Work shift**

---

## 4. Daily-Life Usage

We use **shift** when something changes place or direction.

Examples:

* shift the chair
* shift to another room
* shift your bag
* shift your focus

---

## 5. Academic Usage

In studies, **shift** means:

* change topic
* change method
* move attention

Example:

* The teacher shifted the discussion to AI.

---

## 6. Computer Science Usage

In Computer Science, **shift** has special meanings:

* Moving bits left or right
* Pressing the **Shift** key
* Changing data position
* Moving values in arrays

Example:

* Left shift moves bits to the left.

---

## 7. Beginner Explanation

Think of **shift** as:

➡ “Move something from one place to another.”

It can be:

* physical movement
* data movement
* idea change
* keyboard action

---

## 11. Common Phrases

* shift focus

---

## 12. Memory Technique

Think:

➡ You **shift** a chair from one place to another.

OR

➡ On keyboard, the **Shift** key changes small letters to capital letters.

So, **shift = change or move**.

---

## 13. Common Mistakes Students Make

❌ “I am shift the file.”
✅ “I shifted the file.”

❌ “Shift means only moving objects.”
✅ It can also mean changing ideas, data, or positions.

❌ Confusing **shift** with **switch**
✅ **Shift** = move/change position
✅ **Switch** = replace/change one thing with another

# **Shift** as Different Parts of Speech in Computer Science

---

## 1. **Shift** (Verb)

Meaning: to move data or bits from one position to another.

* The processor shifts the bits to the left during calculation. (Processor calculation ke dauran bits ko left move karta hai)

* The program shifts the array elements automatically. (Program array elements ko automatically move karta hai)

---

## 2. **Shift** (Noun)

Meaning: a movement or change in position/state.

* A left shift increases the binary value. (Left shift binary value ko barha deta hai)

* The system detected a sudden shift in network traffic. (System ne network traffic mein achanak tabdeeli detect ki)

---

## 3. **Shifted** (Verb - Past Form)

Meaning: moved or changed earlier.

* The developer shifted the database to a secure server. (Developer ne database ko secure server par move kiya)

* The data was shifted to another memory location. (Data ko doosri memory location par move kiya gaya)

---

## 4. **Shifting** (Verb - Continuous Form)

Meaning: currently moving or changing.

* The operating system is shifting tasks between CPUs. (Operating system CPUs ke darmiyan tasks move kar raha hai)

* The algorithm is shifting values inside the array. (Algorithm array ke andar values move kar raha hai)

---

## 5. **Shiftable** (Adjective)

Meaning: able to be moved or changed.

* The shiftable data blocks improve memory management. (Shiftable data blocks memory management ko behtar banate hain)

* This variable is shiftable during runtime optimization. (Yeh variable runtime optimization ke dauran move kiya ja sakta hai)
