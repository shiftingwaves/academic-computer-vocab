# Word: **Finite**

## 1. Basic Meaning

**Finite** means:

> “Having a limit or an end.”

Simple English:

> Something that is not endless.

---

## 2. Roman Urdu Meaning

**Finite** =

* *mehdood*
* *limited*
* *jis ka end ho*

Example:

> Human time is finite.
> *(Insani waqt mehdood hota hai.)*

---

## 3. Why This Word Matters

This word is important in:

* Mathematics
* Computer Science
* Logic
* Programming
* Data Structures

You will often hear:

* finite number
* finite memory
* finite state machine

---

## 4. Daily-Life Usage

Example:

> We have finite money for shopping.
> *(Hamare paas shopping ke liye mehdood paise hain.)*

Another:

> Battery life is finite.
> *(Battery ka time limited hota hai.)*

---

## 5. Academic Usage

In studies, “finite” means:

> Limited amount or countable size.

Example:

> The experiment used a finite number of samples.
> *(Experiment mein mehdood samples use hue.)*

---

## 6. Computer Science Usage

In CS, “finite” means:

> A system has limited states, memory, or steps.

Very common term:

> **Finite State Machine (FSM)**

Meaning:

> A machine with limited states.

Example:

> The automaton has finite states.
> *(Automaton ke states mehdood hain.)*

---

## 7. Beginner Explanation

Think like this:

| Thing             | Finite or Infinite? |
| ----------------- | ------------------- |
| 10 apples         | Finite              |
| RAM size          | Finite              |
| Endless sky idea  | Infinite            |
| List with 5 items | Finite              |

So:

> If something can end or can be counted → finite

---

## 8. Synonyms vs Similar Words

| Word      | Meaning             |
| --------- | ------------------- |
| Limited   | mehdood             |
| Bounded   | boundary wala       |
| Countable | gina ja sakta       |
| Finite    | limited with an end |

### Difference:

* **Finite** → has an end
* **Limited** → restricted
* **Countable** → can be counted

## 12. Memory Technique

Remember:

> **Finite = Finish**

Both start with:

> **Fin-**

If something can “finish,” it is finite.

Example:

> A phone battery is finite because it finishes.

---

## 13. Common Mistakes Students Make

### Mistake 1:

❌ Thinking finite means “small”

Wrong.

Finite can be large too.

Example:

> One million is finite.

---

### Mistake 2:

❌ Mixing finite and final

* **Finite** → limited
* **Final** → last

---

### Mistake 3:

❌ Using finite for unlimited things

Wrong:

> “The universe is finite” (unless scientifically proven in context)

Better:

> “The number of students is finite.”
