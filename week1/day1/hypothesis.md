## 📘 Word: **hypothesis**

---

## 1. Basic Meaning

A **hypothesis** is an **idea or guess** that you think might be true, but you need to **test it first**.
(Hypothesis ka matlab ek aisa andaza ya idea jo hum test karke check karte hain)

---

## 2. Roman Urdu Meaning (once)

**andaza / test shuda soch se pehle wala idea**

---

## 3. Why This Word Matters

This word is important because in science and research, we **don’t directly say something is true** — we first make a hypothesis and then test it.

---

## 4. Daily-Life Usage

We use hypotheses when we guess reasons for things, like why something happened.

---

## 5. Academic Usage

In studies, students make hypotheses before doing experiments or research.

---

## 6. Computer Science Usage

In Computer Science, a hypothesis is used when predicting how an algorithm or system will behave.

---

## 7. Beginner Explanation

Think of hypothesis like a **smart guess** that you check later with proof.

---

## 8. Synonyms vs Similar Words

* Guess
* Assumption
* Theory (after testing it becomes stronger)
* Prediction

---

## 9. Antonyms

* Fact
* Proof
* Conclusion
* Certainty

---

## 10. Word Family

* Hypothesis (noun)
* Hypothetical (adjective)
* Hypothesize (verb)
* Hypothetically (adverb)

---

## 11. Common Phrases

* Form a hypothesis
* Test a hypothesis
* Prove a hypothesis
* Reject a hypothesis

---

## 12. Memory Technique

Think: **“Hypo = under, thesis = idea”**
So it is an idea you still need to **prove or test under research**.

---

## 13. Common Mistakes Students Make

* Thinking hypothesis is already a fact (it is NOT proven yet)
* Confusing it with conclusion
* Not testing it properly in experiments

---

# ✍️ 1. Daily-Life Sentences

* I made a hypothesis about the broken fan. (Main ne kharab pankhe ke bare mein andaza lagaya)
* Her hypothesis about rain was correct. (Uska barish ka andaza sahi tha)
* We tested our hypothesis at home. (Hum ne apna andaza ghar par test kiya)

---

# 📚 2. Academic Sentences

* Students write a hypothesis before the experiment. (Students experiment se pehle andaza likhte hain)
* The teacher asked for a clear hypothesis. (Teacher ne ek clear andaza maanga)
* Researchers check if the hypothesis is correct. (Researchers check karte hain ke andaza sahi hai ya nahi)

---

# 💻 3. Computer Science Sentences (ACTIVE VOICE)

* The programmer forms a hypothesis about algorithm performance. (Programmer algorithm ki performance ka andaza banata hai)
* The developer tests a hypothesis using data. (Developer data se andaza test karta hai)
* The scientist uses a hypothesis to design the model. (Scientist model design karne ke liye andaza use karta hai)
* The system generates a hypothesis for user behavior. (System user behavior ka andaza generate karta hai)
* The analyst evaluates a hypothesis about network speed. (Analyst network speed ka andaza evaluate karta hai)

---

# 💻 4. Computer Science Sentences (PASSIVE VOICE)

* A hypothesis is formed by the programmer for testing performance. (Hypothesis programmer ke zariye performance test ke liye banaya jata hai)
* The hypothesis is tested by the developer using real data. (Hypothesis developer ke zariye real data se test kiya jata hai)
* A hypothesis is used by the scientist to design the model. (Hypothesis scientist ke zariye model design karne ke liye use hota hai)
* The hypothesis is generated by the system based on behavior data. (Hypothesis system ke zariye behavior data par banaya jata hai)
* The hypothesis is evaluated by the analyst for network speed. (Hypothesis analyst ke zariye network speed ke liye evaluate kiya jata hai)
