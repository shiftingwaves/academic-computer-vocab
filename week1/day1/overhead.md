## 📘 Word: **overhead**

---

## 1. Basic Meaning

**Overhead** means extra **cost, effort, or resources** needed to run something, not directly useful for the main work.
(Overhead ka matlab extra kharcha ya extra kaam jo directly result nahi deta)

---

## 2. Roman Urdu Meaning (once)

**extra kharcha / extra load / zyada bojh**

---

## 3. Why This Word Matters

We use this word to understand what is **wasted or extra** in time, money, or computing systems.

---

## 4. Daily-Life Usage

Overhead can mean extra cost like electricity bills, rent, or extra effort in daily tasks.

---

## 5. Academic Usage

In studies, overhead refers to extra effort or resources used beyond the main learning task.

---

## 6. Computer Science Usage

In Computer Science, overhead means **extra processing time, memory, or resources** used by a system.

---

## 7. Beginner Explanation

Think of overhead like:
You go to buy one book, but you also pay transport fare — that fare is overhead.

---

## 8. Synonyms vs Similar Words

* Extra cost
* Additional load
* Excess usage
* Waste (in some cases)

---

## 9. Antonyms

* Efficiency
* Saving
* Optimization
* Minimal usage

---

## 10. Word Family

* Overhead (noun/adjective)
* Overhead cost
* Overhead time
* Overhead allocation

---

## 11. Common Phrases

* Reduce overhead
* System overhead
* High overhead
* Overhead cost

---

## 12. Memory Technique

Think: **“Over = extra, Head = burden on top”**
So overhead = **extra burden on top of main work**

---

## 13. Common Mistakes Students Make

* Thinking overhead means only money (it can also be time or memory)
* Confusing it with “main cost” (it is always extra, not main)
* Ignoring overhead in programming performance

---

# ✍️ 1. Daily-Life Sentences

* The delivery service adds overhead charges. (Delivery service extra charges lagata hai)
* Cooking at home reduces overhead costs. (Ghar par khana banana extra kharch kam karta hai)
* Traveling every day creates financial overhead. (Roz safar karna financial extra bojh banata hai)

---

# 📚 2. Academic Sentences

* Schools try to reduce administrative overhead. (Schools administrative extra burden kam karne ki koshish karte hain)
* Group projects sometimes increase overhead for students. (Group projects students ke liye extra load barhate hain)
* Universities analyze overhead costs for better budgeting. (Universities budgeting ke liye extra costs analyze karti hain)

---

# 💻 3. Computer Science Sentences (ACTIVE VOICE)

* The program adds overhead during execution. (Program execution ke dauran extra load add karta hai)
* Developers reduce overhead in the system design. (Developers system design mein extra load kam karte hain)
* The function increases overhead when it runs repeatedly. (Function baar baar chalne par extra load barhata hai)
* The compiler optimizes code to reduce overhead. (Compiler code ko optimize karke extra load kam karta hai)
* The system manages overhead in memory usage. (System memory usage mein extra load manage karta hai)

---

# 💻 4. Computer Science Sentences (PASSIVE VOICE)

* Overhead is added by the program during execution. (Overhead program ke zariye execution ke dauran add hota hai)
* Extra load is reduced by developers in the system design. (Extra load developers ke zariye system design mein kam kiya jata hai)
* Increased overhead is caused by repeated function calls. (Barha hua overhead repeated function calls se hota hai)
* Code is optimized by the compiler to reduce overhead. (Code compiler ke zariye overhead kam karne ke liye optimize hota hai)
* Memory overhead is managed by the system automatically. (Memory overhead system ke zariye automatically manage hota hai)
