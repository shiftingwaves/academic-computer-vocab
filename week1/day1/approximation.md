# Word: **approximation**

---

# 1. Basic Meaning

* **Approximation** = an **estimated value**, not exact
* Jab hum exact answer na dein, balkay **near answer** dein

---

# 2. Roman Urdu Meaning

* **Approximation** = **qareebi andaaza**, **lagbhag sahi value**
* Example:

  * “Approximation of pi”
  * “Pi ka andaza”

---

# 3. Why This Word Matters

* Math aur Computer Science mein bohat important
* Jab exact calculation mushkil ho
* Fast computing ke liye use hota hai

---

# 4. Daily-Life Usage

* “This is an approximation of time.”
* “It will cost about 1000 rupees (approximation).”

---

# 5. Academic Usage

* “We use approximation in calculations.”
* Scientists use it when exact value is difficult

---

# 6. Computer Science Usage

* Algorithms often use **approximation**
* Example:

  * “AI gives approximate results.”
* Graphics:

  * “Rendering uses approximation for speed.”

---

# 7. Beginner Explanation

Socho:

* Exact value = 3.14159265 (pi)
* But we use = 3.14

👉 Ye simple version = **approximation**

---

# 8. Synonyms vs Similar Words

| Word              | Meaning     | Difference        |
| ----------------- | ----------- | ----------------- |
| **approximation** | near value  | formal/scientific |
| **estimate**      | guess value | general use       |
| **guess**         | random idea | less accurate     |
| **close value**   | near result | simple meaning    |

---
