## 📘 Word: **Tolerance**

---

## 1. ⭐ Basic Meaning

**Tolerance** means the ability to allow or accept something different, difficult, or within limits without breaking or failing.
(Yaani kisi cheez ko bardasht karna ya limit ke andar accept karna)

---

## 2. 🇵🇰 Roman Urdu Meaning

**Tolerance = bardasht / had tak ijazat / capacity**
(Kisi cheez ko accept karne ki had ya ability)

---

## 3. 🎯 Why This Word Matters

Tolerance is important because everything has limits—people, machines, and systems.
(Agar limit cross ho jaye to problem ho jati hai)

---

## 4. 🌍 Daily-Life Usage

We use tolerance when we talk about patience, heat, pain, or behavior.
(Rozmarra zindagi mein bardasht aur sabr ka concept hota hai)

---

## 5. 🎓 Academic Usage

In science and engineering, tolerance means acceptable limits of error or variation.
(Academic studies mein allowed error ya range ko tolerance kehte hain)

---

## 6. 💻 Computer Science Usage

In computer science, **tolerance** means how much error, delay, or load a system can handle without failing.
(System kitni ghalti ya load bardasht kar sakta hai)

---

## 7. 🧠 Beginner Explanation

Think of tolerance as a “safe limit.” If you stay inside it, everything works fine.
(Jab tak limit ke andar ho sab theek rehta hai)

---

## 8. 🔁 Synonyms vs Similar Words

**Synonyms:**

* Acceptance
* Patience
* Limit
* Capacity

**Similar Words:**

* Flexibility
* Endurance

---

## 9. ❌ Antonyms

* Intolerance
* Strictness
* Sensitivity
* Fragility

---

## 10. 🧩 Word Family

* Tolerate (verb)
* Tolerant (adjective)
* Tolerance (noun)

---

## 11. 💬 Common Phrases

* error tolerance
* fault tolerance
* heat tolerance
* zero tolerance policy

---

## 12. 🧠 Memory Technique

Think: **“Tolerance = Total limit of acceptance”**
(Jab tak limit ho tab tak system ya insaan manage kar leta hai)

---

## 13. ⚠️ Common Mistakes Students Make

* Confusing tolerance with “patience” only
* Thinking it is only about people (it is also for machines)
* Forgetting it means **limit, not just behavior**
* Using it without context of “acceptable range”

---

# ✍️ SENTENCES

---

## 1. 🌟 Daily-Life Sentences

* I have low tolerance for loud noise. (Mujhe zyada shor bardasht nahi hota)
* Everyone has a different tolerance for pain. (Har insaan ki dard bardasht karne ki capacity different hoti hai)
* This material has high heat tolerance. (Yeh cheez zyada garmi bardasht kar sakti hai)

---

## 2. 🎓 Academic Sentences

* The experiment allows a small tolerance in measurements. (Experiment mein choti ghalti allow hoti hai)
* Engineers calculate tolerance before building machines. (Engineers machine banane se pehle limit set karte hain)
* Scientific data has a fixed tolerance range. (Scientific data ka ek fixed error range hota hai)

---

## 3. 💻 Computer Science Sentences (ACTIVE VOICE)

* The system checks tolerance for network delays. (System network delay ki limit check karta hai)
* Developers set tolerance for memory usage in the program. (Developers memory usage ki limit set karte hain)
* The algorithm measures tolerance for input errors. (Algorithm input errors ki bardasht check karta hai)
* The server handles tolerance for high traffic load. (Server zyada traffic handle karne ki capacity rakhta hai)
* The software defines tolerance for data loss. (Software data loss ki allowed limit define karta hai)

---

## 4. 💻 Computer Science Sentences (PASSIVE VOICE)

* Tolerance is tested by the system during heavy load. (Heavy load par system capacity test hoti hai)
* Tolerance is set by developers for memory limits. (Developers memory ki limit set karte hain)
* Tolerance is measured by the algorithm for errors. (Algorithm errors ki range measure karta hai)
* Tolerance is calculated by the server under stress. (Server pressure mein capacity calculate hoti hai)
* Tolerance is defined by the software for data handling. (Software data handle karne ki limit define karta hai)
