# 📘 Word: **desirable**

## 1. Basic Meaning

**Desirable** means something that people **want**, **like**, or think is **good**.
(Ye aisi cheez hoti hai jo log pasand karte hain ya hasil karna chahte hain)

---

## 2. Roman Urdu Meaning

* Pasandeeda
* Achi aur chahi hui cheez
* Qabil-e-pasand

---

## 3. Why This Word Matters

This word is very common in:

* English speaking
* University studies
* Job interviews
* Computer Science field

For example:

* “Good coding skills are desirable.”
  (Achi coding skills bohat pasand ki jati hain)

---

## 4. Daily-Life Usage

We use **desirable** for:

* Good jobs
* Nice phones
* Good habits
* Popular things

Example:

* A clean room is desirable.
  (Saaf kamra sab ko acha lagta hai)

---

## 5. Academic Usage

In studies, **desirable** means:

* Useful
* Important
* Preferred

Example:

* Good communication skills are desirable for students.
  (Achi communication skills students ke liye bohat zaroori samjhi jati hain)

---

## 6. Computer Science Usage

In Computer Science, **desirable** is used for:

* Good programming skills
* Fast algorithms
* Secure systems
* Clean code

Example:

* A fast algorithm is desirable in software development.
  (Software development mein tez algorithm ko pasand kiya jata hai)

---

## 7. Beginner Explanation

If something is:

* good ✅
* useful ✅
* wanted ✅

then it can be called **desirable**.

Simple idea:

> “People want it.”

---

## 8. Synonyms vs Similar Words

| Word           | Simple Meaning  |
| -------------- | --------------- |
| **Desirable**  | Wanted or liked |
| **Useful**     | Helps in work   |
| **Important**  | Has high value  |
| **Attractive** | Looks nice      |
| **Preferred**  | Chosen more     |

Example:

* A desirable laptop is usually fast and reliable.
  (Behter laptop wo hota hai jo fast aur bharosemand ho)

---

## 9. Antonyms

Opposite words of **desirable**:

* Undesirable
* Unwanted
* Bad
* Unpleasant

Example:

* Slow internet is undesirable.
  (Slow internet kisi ko pasand nahi hota)

---

## 10. Word Family

| Word        | Type               |
| ----------- | ------------------ |
| Desire      | Noun/Verb          |
| Desirable   | Adjective          |
| Desirably   | Adverb             |
| Undesirable | Opposite adjective |

Example:

* Many students desire good grades.
  (Bohat students achay marks chahte hain)

---

## 11. Common Phrases

* Highly desirable
* Desirable qualities
* Desirable outcome
* Desirable skills
* Socially desirable

Example:

* Problem-solving is a highly desirable skill in CS.
  (Problem-solving CS mein bohat important skill mani jati hai)

---

## 12. Memory Technique

### 🧠 Trick:

**Desirable = Desire + able**

If people **desire** something, it is **desirable**.
(Jo cheez log hasil karna chahen usay desirable kehte hain)

---

## 13. Common Mistakes Students Make

❌ “I am desirable this phone.”
✅ “I desire this phone.”
✅ “This phone is desirable.”

❌ Using it for every good thing
✅ Use it for things people strongly want or prefer

# ✍️ 1. Easy Daily-Life Sentences

1. This mobile phone is very desirable.
   (Ye mobile phone logon ko bohat pasand aata hai)

2. Clean clothes are desirable in school.
   (School mein saaf kapray achi baat samjhay jate hain)

3. A quiet place is desirable for study.
   (Parhai ke liye pur-sukoon jagah behtar hoti hai)

---

# 🎓 2. Academic Sentences

1. Good writing skills are desirable for students.
   (Students ke liye achi writing skills zaroori hoti hain)

2. Regular attendance is desirable in class.
   (Class mein roz ana achi aadat mani jati hai)

3. A high GPA is desirable for scholarships.
   (Scholarship ke liye acha GPA bohat important hota hai)

---

# 💻 3. Computer Science Sentences (ACTIVE VOICE)

1. Developers prefer desirable programming features.
   (Developers achi programming features ko zyada pasand karte hain)

2. The company uses desirable security methods.
   (Company behtar security methods istemal karti hai)

3. Students learn desirable coding practices in class.
   (Students class mein achi coding practices seekhte hain)

4. Engineers design desirable software interfaces.
   (Engineers asaan aur behtar software interfaces banate hain)

5. Programmers choose desirable algorithms for speed.
   (Programmers speed ke liye behtar algorithms select karte hain)

---

# 💻 4. Computer Science Sentences (PASSIVE VOICE)

1. Desirable software tools are used in the lab.
   (Lab mein achay software tools istemal kiye jate hain)

2. A desirable database system is selected by the team.
   (Team ki taraf se behtar database system chuna jata hai)

3. Desirable coding standards are followed in the project.
   (Project mein achi coding standards follow ki jati hain)

4. A desirable network design is created for the office.
   (Office ke liye behtar network design tayar kiya jata hai)

5. Desirable testing methods are applied during development.
   (Development ke dauran achay testing methods use kiye jate hain)
