# 📘 Word: **forth**

---

## 1. 🎯 Basic Meaning

**Forth** means **forward or ahead in time, place, or action**.
(It ka matlab hai aagay barhna ya kisi cheez ka samnay aana)

---

## 2. 🗣️ Roman Urdu Meaning

**Aagay / samnay / nikal kar bahar**
(Jab koi cheez forward move kare ya appear ho)

---

## 3. 💡 Why This Word Matters

We use **forth** when we talk about:

* moving forward
* showing something
* giving ideas or results

(It important hai jab hum kisi process ya idea ko aagay le jana batate hain)

---

## 4. 🌍 Daily-Life Usage

We use it in everyday movement or actions.
(Jab hum daily life mein aagay barhna ya nikalna batate hain)

---

## 5. 🎓 Academic Usage

In academic writing, **forth** is used when explaining ideas, arguments, or steps.
(Academic writing mein jab hum points ya ideas present karte hain)

---

## 6. 💻 Computer Science Usage

In Computer Science, **forth** is used when:

* output is produced
* data is generated
* algorithms present results
  (Jab program koi result ya output nikalta hai)

---

## 7. 👶 Beginner Explanation

Think of **forth** as:
👉 “something moving out or forward”
(Jaise koi cheez aagay ya bahar aa rahi ho)

---

## 8. 🔁 Synonyms vs Similar Words

**Synonyms:**

* forward
* ahead
* onward

**Similar words:**

* out
* emerge
  (Ye milte julte words hain jo same idea show karte hain)

---

## 9. ❌ Antonyms

* backward
* behind
* back
  (Is ka opposite peeche jana hota hai)

---

## 10. 📚 Word Family

* forth (adverb)
* forthcoming (coming soon)
* set forth (present something)
* bring forth (produce)

---

## 11. 💬 Common Phrases

* **bring forth ideas**
* **set forth a plan**
* **come forth**
* **go forth**

---

## 12. 🧠 Memory Technique

Think:
👉 “FORTH = FOR + TH = FORWARD”
(Jab bhi FORTH dekho, socho aagay jana)

---

## 13. ⚠️ Common Mistakes Students Make

* Confusing **forth** with **fourth** (number 4th)
* Using it as a noun (it is NOT a noun)
* Writing it alone without context
  (Students aksar isko number samajh lete hain jo ghalat hai)

---

# ✍️ SENTENCES

---

## 1. 🌟 Daily-Life Sentences

* I step forth into the room. (Main kamray mein aagay barh kar jata hoon)
* The child came forth to speak. (Bacha samnay aakar bolne laga)
* She moved forth slowly in the crowd. (Woh bheer mein dheere dheere aagay barhi)

---

## 2. 🎓 Academic Sentences

* The researcher brings forth new ideas. (Researcher naye ideas samnay lata hai)
* The teacher sets forth the rules clearly. (Ustad rules ko wazeh taur par samnay rakhta hai)
* The paper comes forth with strong arguments. (Paper mazboot dalail present karta hai)

---

## 3. 💻 Computer Science Sentences (ACTIVE VOICE)

* The program brings forth the final output. (Program final result nikalta hai)
* The algorithm sets forth the solution steps. (Algorithm solution ke steps present karta hai)
* The system brings forth error messages. (System error messages show karta hai)
* The function sets forth calculated values. (Function calculated values samnay rakhta hai)
* The database brings forth user data. (Database user data nikalta hai)

---

## 4. 🧠 Computer Science Sentences (PASSIVE VOICE)

* The final output is brought forth by the program. (Final result program ke zariye nikalta hai)
* The solution steps are set forth by the algorithm. (Solution steps algorithm ke zariye present hotay hain)
* Error messages are brought forth by the system. (Errors system ke zariye show hotay hain)
* Calculated values are set forth by the function. (Values function ke zariye dikhai jati hain)
* User data is brought forth by the database query. (User data database query se nikalta hai)
