# 📘 Word: **assertion**

## 1. Basic Meaning

An **assertion** is a strong statement that something is true.
(Ek mazboot baat jo kisi cheez ko sach bataye)

---

## 2. Roman Urdu Meaning

**Assertion** =

* “Pakka dawa”
* “Yaqeen se kahi hui baat”
* “Strong statement”
  (Yani jab koi confidence se bole ke yeh baat sahi hai)

---

## 3. Why This Word Matters

This word is important because people use it in:

* arguments
* presentations
* programming
* research
* debugging code

Computer Science students often see **assertions** in programming and testing.
(CS students ko coding aur testing mein yeh lafz bohat milta hai)

---

## 4. Daily-Life Usage

People use **assertion** when:

* someone strongly says something
* a person gives an opinion confidently
* parents or teachers make firm statements

Example:
“His assertion was not accepted.”
(Us ki pakki baat ko logon ne nahi maana)

---

## 5. Academic Usage

In studies, an **assertion** is:

* a claim in essays
* a statement in research
* an idea that needs proof

Example:
“The student’s assertion needs evidence.”
(Student ki baat ko saboot chahiye)

---

## 6. Computer Science Usage

In programming, an **assertion** checks whether something is true in the code.
(Apne code mein check lagana ke cheez sahi chal rahi hai)

Programmers use assertions to:

* find bugs
* test programs
* stop wrong data

Example:

```python
assert x > 0
```

This checks if `x` is greater than 0.
(Yeh check karta hai ke x zero se bara hai)

---

## 7. Beginner Explanation

Think of **assertion** like this:

👉 “I am sure this is correct.”
(Yani mujhe yaqeen hai ke yeh sahi hai)

In coding:
👉 “The program checks if something is correct.”
(Program verify karta hai ke cheez theek hai)

---

## 8. Synonyms vs Similar Words

| Word            | Meaning                      |
| --------------- | ---------------------------- |
| **Claim**       | Saying something is true     |
| **Statement**   | A simple spoken/written idea |
| **Declaration** | Formal statement             |
| **Assertion**   | Strong confident statement   |

---

## 9. Antonyms

| Word            | Meaning                   |
| --------------- | ------------------------- |
| **Denial**      | Saying something is false |
| **Question**    | Doubt about something     |
| **Uncertainty** | Not being sure            |

---

## 11. Common Phrases

* make an assertion
* false assertion
* strong assertion
* assertion error
* assertion testing

---

## 12. Memory Technique

💡 Think:

**Assertion → “I assert this is true!”**
(Yani main yaqeen se keh raha hoon)

Programming memory trick:
👉 “Assertion checks correctness.”
(Assertion code ki sachai check karta hai)

---

## 13. Common Mistakes Students Make

❌ Using “assertion” like a question
✔ Assertion is a confident statement.

❌ Thinking assertion means proof
✔ Assertion is only a claim, not proof.

❌ Confusing “assert” and “assure”
✔ Assert = strongly say
✔ Assure = give confidence

# ✍️ Daily-Life Sentences

1. His assertion surprised everyone.
   (Us ki pakki baat se sab hairan ho gaye)

2. My friend made an assertion about the match result.
   (Mere dost ne match ke result ke bare mein yaqeen se baat ki)

3. The teacher ignored the false assertion.
   (Teacher ne ghalat dawa ko nazarandaz kar diya)

---

# 📚 Academic Sentences

1. The student’s assertion needs evidence.
   (Student ki baat ko saboot chahiye)

2. The article contains a strong assertion.
   (Article mein ek mazboot dawa diya gaya hai)

3. The professor questioned the assertion in the report.
   (Professor ne report ki baat par sawal uthaya)

---

# 💻 Computer Science Sentences (ACTIVE VOICE)

1. The programmer added an assertion to the code.
   (Programmer ne code mein checking statement lagayi)

2. The test checks the assertion before execution.
   (Test chalne se pehle condition verify karta hai)

3. The developer used assertions to find bugs.
   (Developer ne bugs dhoondne ke liye assertions use kiye)

4. The system stops the program after a failed assertion.
   (System ghalat check ke baad program rok deta hai)

5. The software engineer writes assertions during testing.
   (Software engineer testing ke waqt assertions likhta hai)

---

# 💻 Computer Science Sentences (PASSIVE VOICE)

1. The assertion is checked by the compiler.
   (Assertion ko compiler verify karta hai)

2. A warning is shown when the assertion fails.
   (Assertion ghalat hone par warning dikhayi jati hai)

3. Assertions are used in software testing.
   (Assertions software testing mein use hoti hain)

4. The error was detected through an assertion.
   (Error assertion ki madad se pakra gaya)

5. The condition is verified by an assertion.
   (Condition ko assertion check karti hai)
