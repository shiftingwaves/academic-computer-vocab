# 📘 Word: **reside**

## 1. Basic Meaning

**Reside** means **to live in a place** or **stay somewhere permanently**.
(Matlub kisi jagah rehna)

---

## 2. Roman Urdu Meaning

* **Rehna**
* **Kisi jagah basna**
* **Muqeem hona**

---

## 3. Why This Word Matters

This word is common in:

* Forms and documents
* Academic English
* Computer Science discussions
* Official communication

Example:
“Where do you reside?” means “Aap kahan rehte hain?”
(Aap kis jagah rehte hain)

---

## 4. Daily-Life Usage

People use **reside** in formal English instead of “live”.

Example:

* I reside in Islamabad.
  (Main Islamabad mein rehta hoon)

---

## 5. Academic Usage

In universities and books, “reside” sounds more professional.

Example:

* Many students reside in hostels.
  (Kayi students hostel mein rehte hain)

---

## 6. Computer Science Usage

In CS, “reside” often means:

* Data stored in memory
* Files saved in servers
* Programs existing in systems

Example:

* The data resides in the database.
  (Data database mein mojood hai)

---

## 7. Beginner Explanation

Think like this:

👉 **live = simple English**
👉 **reside = formal English**

Both have almost the same meaning.

Example:

* I live in Lahore.
* I reside in Lahore.

## 11. Common Phrases

* **reside in a city**
  (kisi shehar mein rehna)

* **currently reside**
  (is waqt rehna)

* **permanently reside**
  (hamesha ke liye rehna)

* **data resides in memory**
  (data memory mein mojood hai)

---

## 12. Memory Technique

💡 Think:

**“Re-side” → staying on one side/place**

So:
👉 **reside = stay/live in a place**
(Yani kisi jagah tik kar rehna)

---

## 13. Common Mistakes Students Make

❌ “I am reside in Karachi.”
✅ “I reside in Karachi.”

❌ “Data is reside in RAM.”
✅ “Data resides in RAM.”

❌ Using “reside” for temporary visits
✅ Use “stay” for short time

# ✍️ 1. Easy Daily-Life Sentences

1. I reside near the market.
   (Main bazar ke paas rehta hoon)

2. My uncle resides in Lahore.
   (Mere uncle Lahore mein rehte hain)

3. We reside in a small apartment.
   (Hum ek chote flat mein rehte hain)

---

# 📚 2. Academic Sentences

1. Many students reside in university hostels.
   (Kayi students university hostel mein rehte hain)

2. The professor resides near the campus.
   (Professor campus ke qareeb rehte hain)

3. International students reside in separate buildings.
   (Bairooni students alag buildings mein rehte hain)

---

# 💻 3. Computer Science Sentences (ACTIVE VOICE)

1. The files reside on the cloud server.
   (Files cloud server mein save hain)

2. The program resides in system memory during execution.
   (Program chalne ke dauran memory mein hota hai)

3. User data resides in the database.
   (User ka data database mein mojood hota hai)

4. The operating system resides on the hard drive.
   (Operating system hard drive mein install hota hai)

5. The backup files reside in another folder.
   (Backup files doosre folder mein rakhi hoti hain)

---

# 💻 4. Computer Science Sentences (PASSIVE VOICE)

   (Data secure storage mein rakha gaya hai)

   (Aham files remote servers mein mojood rakhi gayi hain)

3. The application is resided in cloud infrastructure.
   (Application cloud system mein mojood hai)

4. The records are resided in encrypted databases.
   (Records encrypted databases mein save kiye gaye hain)

   (Software kai machines mein mojood hai)
