## 📘 Word: **arbitrarily**

---

## 1. 🎯 Basic Meaning

**Arbitrarily** means doing something **without a clear reason, rule, or system**.
(Arbitrarily ka matlab hai koi kaam bina reason, rule ya system ke karna)

---

## 2. 🟢 Roman Urdu Meaning

**Arbitrarily = bina soch samajh / bina rule ke / random tareeqay se**
(Jab koi decision ya kaam bina kisi proper reason ke ho)

---

## 3. ⭐ Why This Word Matters

This word is important because in study, science, and programming we must avoid random decisions.
(Yeh word important hai kyun ke humein system ke baghair decisions nahi lene chahiye)

---

## 4. 🏠 Daily-Life Usage

We use “arbitrarily” when someone chooses something randomly without logic.
(Daily life mein jab koi banda bina wajah kuch select kare)

---

## 5. 🎓 Academic Usage

In academic writing, it is used when results or choices are not based on rules or logic.
(Academic writing mein jab koi cheez logic ya rule ke baghair ho)

---

## 6. 💻 Computer Science Usage

In CS, “arbitrarily” means values or choices that are **random or not fixed**, often used in algorithms or inputs.
(CS mein arbitrarily ka matlab hota hai random ya fixed rule ke baghair value lena)

---

## 7. 🧠 Beginner Explanation

Think of it like:
👉 “I picked it randomly, no reason at all.”
(Arbitrarily = main ne bina sochay samjhay choose kiya)

---

## 8. 🔁 Synonyms vs Similar Words

**Synonyms:**

* randomly
* without reason
* haphazardly

**Similar Words:**

* casually
* loosely
* blindly

---

## 9. ❌ Antonyms

* logically
* systematically
* carefully
* intentionally

---

## 10. 📚 Word Family

* arbitrary (adjective)
* arbitrarily (adverb)
* arbitrariness (noun)

---

## 11. 💬 Common Phrases

* arbitrarily chosen
* arbitrarily assigned
* arbitrarily selected values
* arbitrarily defined rules

---

## 12. 🧠 Memory Technique

Think: **“ARBITRARY = NO RULE”**
(Arbitrary matlab koi rule follow nahi ho raha)

---

## 13. ⚠️ Common Mistakes Students Make

* Thinking it means “important” (it actually means random)
* Using it for planned actions (it is only for unplanned/no-rule actions)
* Confusing it with “optional” (it is not optional, it is random)

# ✏️ Sentences

---

## 🏠 1. Daily-Life Sentences (3)

* He arbitrarily chose a movie to watch. (Us ne bina sochay samjhay movie choose ki)
* The teacher arbitrarily picked a student. (Teacher ne random student select kiya)
* She arbitrarily changed her decision. (Us ne bina reason apna faisla badal diya)

---

## 🎓 2. Academic Sentences (3)

* The data was arbitrarily divided into groups. (Data ko bina system ke groups mein divide kiya gaya)
* The rule was arbitrarily applied by the committee. (Rule committee ne bina logic ke apply kiya)
* The samples were arbitrarily selected for the study. (Samples study ke liye random choose kiye gaye)

---

## 💻 3. Computer Science Sentences (Active Voice) (5)

* The program arbitrarily selects a value from the array. (Program array se random value select karta hai)
* The algorithm arbitrarily assigns IDs to users. (Algorithm users ko random IDs assign karta hai)
* The system arbitrarily generates test data. (System random test data generate karta hai)
* The developer arbitrarily sets default parameters. (Developer random default parameters set karta hai)
* The function arbitrarily picks a node in the graph. (Function graph mein random node select karta hai)

---

## 💻 4. Computer Science Sentences (Passive Voice) (5)

* Values are arbitrarily chosen by the algorithm. (Values algorithm ke zariye random choose hoti hain)
* Test cases are arbitrarily generated by the system. (Test cases system ke zariye random banaye jate hain)
* IDs are arbitrarily assigned to new users. (IDs naye users ko random assign ki jati hain)
* Nodes are arbitrarily selected in the graph process. (Graph process mein nodes random select hoti hain)
* Parameters are arbitrarily set in the configuration. (Configuration mein parameters bina rule ke set hotay hain)
