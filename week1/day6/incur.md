# 📘 Word: **incur**

## 1. Basic Meaning

**Incur** means:
👉 to experience something unwanted
👉 to receive a loss, cost, punishment, or problem because of an action

(Roman Urdu: **kisi nuksan, kharch ya mushkil ka samna karna**)

---

## 2. Roman Urdu Meaning

**Incur** =
👉 **bardasht karna**
👉 **nuksan uthana**
👉 **kharcha ya saza lena**

---

## 3. Why This Word Matters

This word is common in:

* academic English
* business
* finance
* Computer Science documentation

You may hear:

* incur costs
* incur losses
* incur penalties

(Roman Urdu: **yeh lafz zyada tar tab use hota hai jab kisi action ki wajah se nuksan ya extra cheez mile**)

---

## 4. Daily-Life Usage

We use **incur** when someone gets an unwanted result.

Example:

* He incurred extra charges.
  (**us par extra charges lag gaye**)

---

## 5. Academic Usage

In studies and reports, this word is used for:

* expenses
* penalties
* risks
* losses

Example:

* The company incurred heavy losses.
  (**company ko bara nuksan hua**)

---

## 6. Computer Science Usage

In Computer Science, **incur** is used for:

* memory cost
* processing cost
* network delay
* performance overhead

Example:

* The algorithm incurs high memory usage.
  (**algorithm zyada memory use karta hai**)

---

## 7. Beginner Explanation

Think like this:

If your action causes a problem or cost,
👉 you **incur** it.

Example:

* Late submission → incur penalty

(Roman Urdu: **agar kisi kaam ki wajah se nuksan ya extra kharcha aaye to usay incur kehte hain**)

---

## 8. Synonyms vs Similar Words

| Word           | Meaning                    |
| -------------- | -------------------------- |
| **incur**      | receive something unwanted |
| **suffer**     | feel pain or loss          |
| **experience** | go through something       |
| **obtain**     | get something              |
| **face**       | deal with something        |

Example:

* The server incurred a delay.
  (**server ko delay ka samna karna para**)

---

## 9. Antonyms

Opposite words:

* avoid
* prevent
* escape

Example:

* The company avoided extra costs.
  (**company ne extra kharchay se bachao kiya**)

---

## 10. Word Family

| Word      | Type            |
| --------- | --------------- |
| incur     | verb            |
| incurred  | past form       |
| incurring | continuous form |

Example:

* The system incurred errors during testing.
  (**testing ke dauran system mein errors aaye**)

---

## 11. Common Phrases

* incur a loss
* incur costs
* incur penalties
* incur expenses
* incur damage

Example:

* Users may incur extra fees.
  (**users par extra fees lag sakti hai**)

---

## 12. Memory Technique

💡 Memory Trick:

Think:
👉 **Incur = Invite Cost Unwanted Result**

“IN-CUR” → “inside comes a problem”

(Roman Urdu: **incur ko yaad rakho jab kisi action ke baad nuksan aaye**)

---

## 13. Common Mistakes Students Make

❌ Wrong:

* “I incurred a gift.”

✅ Correct:

* “I incurred a penalty.”

👉 **Incur** is mostly used for negative things.

# ✍️ 1. Easy Daily-Life Sentences

* We incurred extra travel costs yesterday.
  (**kal humein safar ka extra kharcha hua**)

* He incurred a fine for late payment.
  (**late payment ki wajah se us par jurmana laga**)

* They incurred problems during the trip.
  (**safar mein unhein mushkilaat ka samna hua**)

---

# ✍️ 2. Academic Sentences

* The project incurred additional expenses.
  (**project mein extra kharcha aaya**)

* The student incurred a penalty for cheating.
  (**cheating ki wajah se student ko saza mili**)

* The experiment incurred unexpected risks.
  (**experiment mein unexpected risks saamne aaye**)

---

# ✍️ 3. Computer Science Sentences (ACTIVE VOICE)

* The algorithm incurs high processing costs.
  (**algorithm zyada processing cost leta hai**)

* The application incurs memory overhead.
  (**application extra memory use karti hai**)

* The network incurs delays during heavy traffic.
  (**heavy traffic mein network slow ho jata hai**)

* The database incurs storage costs over time.
  (**database waqt ke sath storage ka kharcha barhata hai**)

* The software incurs errors after the update.
  (**update ke baad software mein errors aate hain**)

---

# ✍️ 4. Computer Science Sentences (PASSIVE VOICE)

* Extra memory costs were incurred by the system.
  (**system ki wajah se extra memory cost aayi**)

* A delay was incurred during data transfer.
  (**data transfer mein delay aya**)

* High processing costs were incurred by the server.
  (**server ki wajah se processing cost zyada hui**)

* Additional storage expenses were incurred during backup.
  (**backup ke dauran extra storage ka kharcha hua**)

* Network latency was incurred during online communication.
  (**online communication mein network delay hua**)
