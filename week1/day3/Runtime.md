# **Runtime**

## 1. Basic Meaning

**Runtime** means the time when a program is **running or executing**.
(Matlab jab program chal raha hota hai us waqt ko runtime kehte hain)

---

## 2. Roman Urdu Meaning

* **Program chalne ka waqt**
* **Execution time**
* **Run hone ka time**

---

## 3. Why This Word Matters

This word is very important in:

* Programming
* Software development
* Algorithms
* Debugging
* Performance testing

CS students often hear:

* runtime error
* runtime environment
* runtime performance

(Yeh word programming mein bohat common hai)

---

## 4. Daily-Life Usage

In normal life, runtime can mean:

* how long something works
* active working time

Example:

* The battery runtime is four hours.
  (Battery 4 ghante tak chalti hai)

---

## 5. Academic Usage

In studies, runtime is used for:

* experiments
* software analysis
* algorithm efficiency

Example:

* We measured the runtime of the algorithm.
  (Hum ne algorithm ka chalne ka time check kiya)

---

## 6. Computer Science Usage

In CS, **runtime** usually means:

* the period when code is executing
* the execution speed of a program
* errors that happen while running

Example:

* A runtime error stopped the program.
  (Program chalte waqt error aa gaya)

---

## 7. Beginner Explanation

When you click **Run** in a program:

* the code starts working
* memory is used
* instructions execute

This period is called **runtime**.
(Yani jab code asal mein chal raha ho to usay runtime kehte hain)

---

## 8. Synonyms vs Similar Words

| Word                | Meaning                |
| ------------------- | ---------------------- |
| **Execution time**  | program chalne ka waqt |
| **Processing time** | processing duration    |
| **Run duration**    | total running time     |

### Small Difference

* **Runtime** = while program is running
* **Compile time** = before program runs

---

## 9. Antonyms

* **Compile time**
* **Idle time**
* **Stopped state**

Example:

* Compile time checks happen before runtime.
  (Runtime se pehle code check hota hai)

---

---

## 12. Memory Technique

Think:

> **Runtime = Run + Time**

If the program is running, that time is runtime.
(Jab program chal raha ho to woh runtime hota hai)

---

## 13. Common Mistakes Students Make

### ❌ Wrong:

* Runtime means “coding time”

### ✔ Correct:

* Runtime means “program execution time”

---

### ❌ Wrong:

* Runtime and compile time are the same.

### ✔ Correct:

* Compile time happens before execution.
* Runtime happens during execution.
  (Ek code chalne se pehle hota hai aur doosra chalne ke duran)
