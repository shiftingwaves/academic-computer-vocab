# 📘 Word: **artifact**

## 1. Basic Meaning

An **artifact** is an object, mark, file, or result that was made by humans or created during a process.
(Ye insan ya process ki banayi hui cheez hoti hai)

---

## 2. Roman Urdu Meaning

* **Purani ya banayi hui cheez**
* **Process ka result**
* **Extra nishaan ya file**

Example meaning in CS:
A compiled file can be called an artifact.
(Program compile hone ke baad jo file banti hai usay artifact kehte hain)

---

## 3. Why This Word Matters

The word **artifact** is important because:

* It is used in **history**
* It is used in **science**
* It is very common in **Computer Science**
* Developers use this word in software building and testing

(Ye word academics aur software field dono mein bohat use hota hai)

---

## 4. Daily-Life Usage

People use **artifact** for:

* old objects
* museum items
* handmade things
* unwanted marks in photos/videos

(Museum ki purani cheezen bhi artifacts hoti hain)

---

## 5. Academic Usage

In academics, an artifact can mean:

* a historical object
* research material
* a project result

(Taleemi kaam mein kisi project ya history object ko artifact kaha ja sakta hai)

---

## 6. Computer Science Usage

In Computer Science, **artifact** often means:

* compiled files
* generated reports
* build outputs
* logs
* test results

(Software process ke baad jo output files banti hain unhein artifacts kehte hain)

---

## 7. Beginner Explanation

Think like this:

👉 You write code
👉 The compiler creates a `.exe` file
👉 That output file is an **artifact**

(Simple process ke baad jo result banta hai usay artifact kehte hain)

---

## 8. Synonyms vs Similar Words

| Word         | Meaning                       |
| ------------ | ----------------------------- |
| **Object**   | Any thing                     |
| **Item**     | A single thing                |
| **Output**   | Result of a process           |
| **File**     | Digital document              |
| **Artifact** | Special created result/object |

---

## 9. Antonyms

* natural object
* original source
* raw input

(Yani aisi cheez jo process ke baad na bani ho)

---

(Ye phrases CS aur academics mein common hain)

---

## 12. Memory Technique

### 🧠 Memory Trick:

**Artifact = Art + Fact**

Imagine:
A person creates something with skill → that created thing becomes an **artifact**.

(Jo cheez insan banaye ya process se bane usay artifact yaad rakho)

---

## 13. Common Mistakes Students Make

❌ Thinking artifact only means museum objects
✅ In CS, it also means software outputs

❌ Confusing artifact with “artificial”
✅ Artifact = created object/result

❌ Using it only for physical objects
✅ Digital files can also be artifacts
