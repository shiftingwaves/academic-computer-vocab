# **Constrain**

## 1. Basic Meaning

**Constrain** means to **limit**, **control**, or **stop something from becoming too much**.
(Matlab kisi cheez ko hadd mein rakhna ya control karna)

---

## 2. Roman Urdu Meaning

* **Limit karna**
* **Control mein rakhna**
* **Rokna ya bandish lagana**

---

## 3. Why This Word Matters

This word is very common in:

* Programming
* Databases
* Algorithms
* Mathematics
* Research papers

Computer Science students often see:

* **constraints**
* **memory constraints**
* **time constraints**
* **data constraints**

(Is liye yeh word CS studies mein bohat use hota hai)

---

## 4. Daily-Life Usage

We use **constrain** when something is limited.

Example:

* Money can constrain shopping.
  (Paise kam hon to shopping limited ho jati hai)

* Time can constrain our plans.
  (Waqt ki kami plans ko rok deti hai)

---

## 5. Academic Usage

In studies, **constrain** means:

* limiting choices
* controlling conditions
* setting rules

Example:

* The teacher constrained the project size.
  (Teacher ne project ki limit rakh di)

---

## 6. Computer Science Usage

In CS, **constrain** is used for:

* limiting memory
* restricting input values
* controlling system behavior
* database rules

Example:

* Constraints protect the database from wrong data.
  (Database ko ghalat data se bachaya jata hai)

---

## 7. Beginner Explanation

Imagine you have only:

* 1 GB RAM
* 1 hour
* small storage

These limits **constrain** your program.

(Yani resources kam hon to kaam limited ho jata hai)

---

## 8. Synonyms vs Similar Words

| Word         | Meaning           |
| ------------ | ----------------- |
| **Limit**    | hadd lagana       |
| **Restrict** | rokna             |
| **Control**  | qaboo mein rakhna |
| **Prevent**  | hone se rokna     |

### Small Difference

* **Constrain** = limit because of rules/resources
* **Restrict** = strongly stop something

---

## 9. Antonyms

* **Allow**
* **Free**
* **Expand**
* **Release**

Example:

* More memory frees the program.
  (Zyada memory program ko asani deti hai)

---

## 10. Word Family

| Word             | Type                |
| ---------------- | ------------------- |
| **Constrain**    | Verb                |
| **Constraint**   | Noun                |
| **Constrained**  | Adjective           |
| **Constraining** | Adjective/Verb form |

Example:

* Memory constraint
  (Memory ki limit)

---

## 11. Common Phrases

* **Time constraints**
* **Memory constraints**
* **Resource constraints**
* **Constrain the input**
* **Constrained environment**

---

## 12. Memory Technique

Think:

> **“Constrain = chain”**

A chain limits movement.
Same way, constraints limit actions.
(Jaise chain movement rok deti hai, waise constrain bhi limit karta hai)

---

## 13. Common Mistakes Students Make

### ❌ Wrong:

* Constrain means “destroy”

### ✔ Correct:

* Constrain means “limit” or “control”

---

### ❌ Wrong:

* “I am constrain.”

### ✔ Correct:

* “I am constrained by time.”
  (Main waqt ki kami ki wajah se limited hoon)

# 🌍 3 Easy Daily-Life Sentences

1. My small budget constrains my shopping.
   (Mere paas kam paise hain is liye shopping limited rehti hai)

2. Heavy traffic constrained our travel time.
   (Traffic ki wajah se hum der se pohanche)

3. Family rules constrain late-night gaming.
   (Ghar ke rules ki wajah se zyada der game nahi khel sakte)

---

# 📚 3 Academic Sentences

1. The exam time constrained the students.
   (Imtihan ka kam waqt students ke liye mushkil bana)

2. The teacher constrained the project length.
   (Teacher ne project ki limit rakh di)

3. Limited resources constrain scientific research.
   (Kam resources research ko mushkil bana dete hain)

---

# 💻 5 Computer Science Sentences (ACTIVE VOICE)

1. The programmer constrained the input values in the form.
   (Programmer ne form mein sirf limited values allow ki)

2. The operating system constrains memory usage for apps.
   (System apps ki memory usage ko control karta hai)

3. The developer constrained file uploads to 5 MB.
   (Developer ne file size ki limit 5 MB rakhi)

4. The algorithm constrains unnecessary calculations.
   (Algorithm extra calculations ko kam karta hai)

5. The database administrator constrained duplicate entries.
   (Database admin ne same data dobara save hone se roka)

---

# 💻 5 Computer Science Sentences (PASSIVE VOICE)

1. User input is constrained by validation rules.
   (User ki entry rules ke mutabiq limited rakhi jati hai)

2. Memory usage is constrained in embedded systems.
   (Embedded systems mein memory carefully limited hoti hai)

3. The search results were constrained by filters.
   (Filters ki wajah se limited results dikhaye gaye)

4. Network access is constrained by security policies.
   (Security rules ki wajah se network access control hota hai)

5. File sizes are constrained by the server configuration.
   (Server settings ki wajah se file size limited hota hai)
