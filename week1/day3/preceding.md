## 📘 Word: **preceding**

---

## 1. 🧠 Basic Meaning

**Preceding** means **coming before something else in time, order, or position**.
(Preceding ka matlab hota hai kisi cheez se pehle aana ya pehle hona)

---

## 2. 🇵🇰 Roman Urdu Meaning

**Preceding = pehle wala / jo kisi cheez se pehle ho**
(yeh us cheez ko show karta hai jo baad wali cheez se pehle ho)

---

## 3. ⭐ Why This Word Matters

In studies and Computer Science, order is very important.
We often need to know what happened **before** another step.
(Computer Science aur studies mein sequence bohat important hota hai)

---

## 4. 🌍 Daily-Life Usage

We use “preceding” when talking about events, time, or actions.
(rozmarra zindagi mein hum pehle wali cheez batane ke liye use karte hain)

---

## 5. 🎓 Academic Usage

In academic writing, “preceding” is used to refer to earlier ideas or paragraphs.
(academic writing mein pehle wali explanation ya paragraph ke liye use hota hai)

---

## 6. 💻 Computer Science Usage

In CS, “preceding” refers to previous steps, nodes, or instructions in a process.
(Computer Science mein pehle wale steps ya data ko show karta hai)

---

## 7. 👶 Beginner Explanation

Think of a line of students. The student standing before you is your **preceding student**.
(socho line mein jo tum se pehle khara ho woh preceding hota hai)

---

## 8. 🔁 Synonyms vs Similar Words

* **Synonyms:** previous, prior, earlier
* **Similar words:** before, former
  (sab ka idea hota hai “pehle wala”)

---

## 9. ❌ Antonyms

* following
* next
* subsequent
  (yeh sab “baad mein aane wala” show karte hain)

---

## 10. 📚 Word Family

* precede (verb)
* precedent
* precedence
* preceding
  (yeh sab words “pehle aana” ya “order” se related hain)

---

## 11. 💬 Common Phrases

* preceding step
* preceding event
* preceding paragraph
* preceding node
* preceding instruction

---

## 12. 🧠 Memory Technique

Think: **PRE = before**
So “preceding” = “pre + something = before something”
(“pre” yaad rakho = pehle)

---

## 13. ⚠️ Common Mistakes Students Make

* Confusing “preceding” with “proceeding”
* Thinking it means “continuing”
* Using it for future instead of past order
  (yeh galti hoti hai ke students isay aglay ke liye samajh lete hain)

---

# ✍️ SENTENCES

---

## 1. 🌍 Daily-Life Sentences

* The preceding day was very cold. (Pichla din bohat thanda tha)
* She checked the preceding message carefully. (Us ne pehla message dhyan se check kiya)
* The preceding train arrived late. (Pichli train late aayi)

---

## 2. 🎓 Academic Sentences

* The preceding chapter explains basic concepts. (Pichla chapter basic concepts samjhata hai)
* The preceding results are important for analysis. (Pichle results analysis ke liye important hain)
* The preceding paragraph gives background information. (Pichla paragraph background information deta hai)

---

## 3. 💻 Computer Science Sentences (ACTIVE VOICE)

* The program reads the preceding input value. (Program pehle input value read karta hai)
* The function processes the preceding data block. (Function pehle data block process karta hai)
* The algorithm compares the preceding node with the current node. (Algorithm pehle node ko current node se compare karta hai)
* The system stores the preceding user request in memory. (System pehle user request memory mein store karta hai)
* The compiler checks the preceding line of code for errors. (Compiler pehli code line ko errors ke liye check karta hai)

---

## 4. 💻 Computer Science Sentences (PASSIVE VOICE)

* The preceding value is read by the program. (Pehli value program ke zariye read hoti hai)
* The preceding data block is processed by the function. (Pehla data block function ke zariye process hota hai)
* The preceding node is compared with the current node by the algorithm. (Pehla node algorithm ke zariye compare hota hai)
* The preceding request is stored in memory by the system. (Pehli request system ke zariye memory mein store hoti hai)
* The preceding line of code is checked for errors by the compiler. (Pehli code line compiler ke zariye errors ke liye check hoti hai)
