## 📘 Word: **Inequality**

---

## 1. 🧠 Basic Meaning

An **inequality** shows that two things are **not equal**.
It compares values using symbols like **<, >, ≤, ≥**.
([Iska matlab hai do cheezen barabar nahi hoti])

---

## 2. 🗣️ Roman Urdu Meaning

**Inequality = barabari na hona / do cheezon ka unequal hona**
([Jab do values ya cheezen same na hon])

---

## 3. ⭐ Why This Word Matters

We use inequality to compare numbers, values, and conditions.
It helps us understand limits and rules.
([Yeh hume batata hai ke kaunsi value zyada ya kam hai])

---

## 4. 🏠 Daily-Life Usage

We use inequality in real life when comparing money, age, or time.
([Rozmarra zindagi mein comparison ke liye use hota hai])

---

## 5. 🎓 Academic Usage

In math, inequalities help solve problems with ranges instead of exact answers.
([Math mein exact number ke bajaye range batane ke liye use hota hai])

---

## 6. 💻 Computer Science Usage

In Computer Science, inequalities are used in **conditions**, **loops**, and **algorithms**.
([Programming mein decisions banane ke liye use hota hai])

---

## 7. 👶 Beginner Explanation

If two values are not equal, we use inequality symbols like > or <.
([Agar do values same na hon to inequality use hoti hai])

---

## 8. 🔁 Synonyms vs Similar Words

* **Not equal**
* **Difference**
* **Comparison**
  ([Yeh words similar idea dete hain ke cheezen same nahi hain])

---

## 9. ⛔ Antonyms

* Equality (barabari)
* Same
  ([Jab dono cheezen bilkul equal hon])

---

## 10. 📚 Word Family

* Equal
* Inequality
* Equalize
* Unequal
  ([Sab words barabari ya difference se related hain])

---

## 11. 💬 Common Phrases

* greater than inequality
* less than inequality
* solve inequality
  ([Math aur programming mein common phrases])

---

## 12. 🧠 Memory Technique

Think: **“IN + EQUAL = NOT EQUAL”**
([Socho IN ka matlab andar nahi, yani equal nahi])

---

## 13. ❌ Common Mistakes Students Make

* Confusing **>** and **<**
* Thinking inequality means always “wrong”
* Forgetting that inequality shows **range**, not exact value
  ([Students symbols aur meaning confuse kar lete hain])

---

# ✍️ SENTENCES

---

## 🏠 1. Daily-Life Sentences

* My salary is greater than my friend’s salary. (Meri salary mere dost se zyada hai)
* His age is less than my age. (Uski age meri age se kam hai)
* I spend more than I earn sometimes. (Kabhi kabhi main zyada kharch kar deta hoon)

---

## 🎓 2. Academic Sentences

* The inequality shows the relationship between two numbers. (Inequality do numbers ka comparison batati hai)
* We solve inequalities to find possible values of x. (Hum x ki possible values nikalne ke liye inequalities solve karte hain)
* The teacher explained linear inequalities in class. (Teacher ne class mein linear inequalities samjhai)

---

## 💻 3. Computer Science Sentences (ACTIVE VOICE)

* The program checks inequality between two inputs. (Program do inputs ka comparison check karta hai)
* The algorithm uses inequality to make decisions. (Algorithm decisions lene ke liye inequality use karta hai)
* The system compares values using inequality operators. (System values compare karne ke liye inequality operators use karta hai)
* The code filters data using inequality conditions. (Code inequality conditions se data filter karta hai)
* The function evaluates inequality in the loop. (Function loop mein inequality evaluate karta hai)

---

## 💻 4. Computer Science Sentences (PASSIVE VOICE)

* Inequality is checked between two inputs by the program. (Do inputs ka comparison program ke through check hota hai)
* Decisions are made using inequality in the algorithm. (Algorithm mein decisions inequality se liye jate hain)
* Values are compared using inequality operators by the system. (System ke zariye values inequality operators se compare hoti hain)
* Data is filtered using inequality conditions in the code. (Code mein data inequality conditions se filter hota hai)
* Inequality is evaluated in the loop by the function. (Function ke through loop mein inequality evaluate hoti hai)
