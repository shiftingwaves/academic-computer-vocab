## 📘 Word: **Outlier**

---

## 1. 🧠 Basic Meaning

An **outlier** is a value or thing that is **very different from others in a group**. (Yani aisi cheez jo baqi sab se kaafi different ho)

---

## 2. 🇵🇰 Roman Urdu Meaning

**Outlier = aisa data point ya cheez jo group se bilkul different ho** (Jo baqi sab ke beech fit na ho)

---

## 3. ⭐ Why This Word Matters

We use “outlier” to find **unusual or strange values** in data. (Yani jo cheez normal pattern se bahar ho)

---

## 4. 🌍 Daily-Life Usage

In real life, an outlier is something that is **not like others in a group**. (Yani jo cheez baqi logon ya cheezon jaisi na ho)

---

## 5. 🎓 Academic Usage

In studies, an outlier is a **data value that is far away from other values in a dataset**. (Yani research mein jo number ya result normal se different ho)

---

## 6. 💻 Computer Science Usage

In CS, an outlier is a **data point that does not follow the pattern of the dataset**. (Yani dataset ka aisa value jo pattern break kare)

---

## 7. 👶 Beginner Explanation

Think of outlier like a **student who gets 100 marks while everyone else gets around 50**. (Yani aik banda jo sab se bohat different result de)

---

## 8. 🔁 Synonyms vs Similar Words

**Synonyms:**

* Exception
* Irregular value
* Anomaly

**Similar Words (but not same):**

* Error (galti hoti hai, outlier zaroori nahi galat ho)
* Noise (random disturbance hota hai)

---

## 9. ❌ Antonyms

* Normal value
* Average
* Typical value
* Common data point

---

## 10. 📚 Word Family

* Outlier (noun)
* Outlying (adjective)
* Outlie (rare verb form, not commonly used)

---

## 11. 💬 Common Phrases

* Data outlier
* Statistical outlier
* Remove outliers
* Detect outliers
* Extreme outlier

---

## 12. 🧠 Memory Technique

Think: **“OUT = outside, LIER = lying outside group”**
So outlier = jo group ke bahar ho (Yani jo baqi data se bahar nikal jaye)

---

## 13. ⚠️ Common Mistakes Students Make

* Thinking outlier means “error” (ye zaroori nahi)
* Ignoring outliers in data analysis
* Confusing outlier with average values
* Using it for people instead of data context

---

# ✍️ SENTENCES

---

## 1. 🏠 3 Easy Daily-Life Sentences

* He is an outlier in his family because he loves coding. (Woh apne ghar mein different hai kyun ke usay coding pasand hai)
* This tree is an outlier in the garden because it is very tall. (Yeh darakht baqi sab se zyada lamba hai)
* My score is an outlier compared to my classmates. (Mera score baqi class se bilkul different hai)

---

## 2. 🎓 3 Academic Sentences

* The researcher identified an outlier in the dataset. (Researcher ne dataset mein aik unusual value dhoondi)
* An outlier can affect the results of a study. (Outlier study ke results ko effect kar sakta hai)
* The teacher explained how to handle an outlier in statistics. (Teacher ne bataya ke statistics mein outlier kaise handle karte hain)

---

## 3. 💻 5 Computer Science Sentences (ACTIVE VOICE)

* The program detects an outlier in the data set. (Program data mein unusual value detect karta hai)
* The algorithm removes outliers from the dataset. (Algorithm dataset se unusual values remove karta hai)
* The system flags an outlier during analysis. (System analysis ke dauran outlier mark karta hai)
* Data scientists study an outlier carefully. (Data scientists outlier ko dhyan se study karte hain)
* The model identifies an outlier in user behavior. (Model user behavior mein unusual pattern identify karta hai)

---

## 4. 💻 5 Computer Science Sentences (PASSIVE VOICE)

* An outlier is detected by the machine learning model. (Outlier machine learning model ke zariye detect hota hai)
* Outliers are removed by the preprocessing algorithm. (Outliers preprocessing algorithm ke zariye remove hote hain)
* A suspicious value is marked as an outlier by the system. (Aik suspicious value ko system outlier mark karta hai)
* Outliers are analyzed by data scientists. (Outliers ko data scientists analyze karte hain)
* The dataset is cleaned by removing outliers. (Dataset ko outliers remove karke clean kiya jata hai)
