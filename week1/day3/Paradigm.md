# 📘 Word: **Paradigm**

---

## 1. 🧠 Basic Meaning

A **paradigm** is a **model, pattern, or way of thinking** that people follow.
(Paradigm ek aisa model ya soch ka tareeqa hota hai jo log follow karte hain)

---

## 2. 🇵🇰 Roman Urdu Meaning

**Paradigm = ek standard model ya way of thinking / naya system ya approach**
(Paradigm matlab koi fixed pattern ya naya tareeqa jisse hum kaam karte hain ya sochte hain)

---

## 3. ⭐ Why This Word Matters

In learning and science, a **paradigm** helps us understand how something works in a structured way.
(Ye word is liye important hai kyun ke ye batata hai ke hum kisi cheez ko kis pattern ya system ke through samjhte hain)

---

## 4. 🌍 Daily-Life Usage

We use paradigms when we talk about habits or ways of doing things.
(Paradigm daily life mein kisi kaam ko karne ka fixed tareeqa hota hai)

---

## 5. 🎓 Academic Usage

In studies, a paradigm is a **framework for understanding knowledge or theories**.
(Academic level par paradigm ek soch ka framework hota hai jo learning ko structure deta hai)

---

## 6. 💻 Computer Science Usage

In Computer Science, a paradigm is a **programming style or approach** like object-oriented or functional programming.
(CS mein paradigm programming ka tareeqa hota hai jese object-oriented ya functional programming)

---

## 7. 👶 Beginner Explanation

Think of a paradigm as a **rulebook or style guide** that tells you how to think or work.
(Paradigm samjho jese ek guidebook jo batata hai kaam ka sahi tareeqa)

---

# ✍️ Sentences

---

## 1. 🏡 Daily-Life Sentences

* I follow a new paradigm for studying daily. (Main parhne ke liye naya tareeqa follow karta hoon)
* This school uses a strict paradigm for discipline. (Is school mein discipline ka ek fixed system hota hai)
* My daily routine changed my learning paradigm. (Meri routine ne mera parhne ka tareeqa badal diya)

---

## 2. 🎓 Academic Sentences

* The teacher explained a scientific paradigm in class. (Ustaad ne class mein scientific model samjhaya)
* Students study different learning paradigms in education. (Students mukhtalif parhne ke systems study karte hain)
* The research is based on a strong theoretical paradigm. (Research ek strong theory ke framework par based hai)

---

## 3. 💻 Computer Science Sentences (ACTIVE VOICE)

* Developers design software using an object-oriented paradigm. (Developers software object-oriented tareeqe se banate hain)
* The programmer applies a functional programming paradigm. (Programmer functional programming style use karta hai)
* Students learn how paradigms affect coding structure. (Students seekhte hain ke paradigms coding ko kaise shape karte hain)
* Engineers choose the best paradigm for system design. (Engineers system design ke liye best method choose karte hain)
* The team compares different programming paradigms. (Team different programming styles compare karti hai)

---

## 4. 💻 Computer Science Sentences (PASSIVE VOICE)

* A new programming paradigm is introduced in the course. (Course mein naya programming method introduce hota hai)
* Object-oriented paradigm is widely used in software development. (Object-oriented method software development mein bohat use hota hai)
* Different paradigms are taught in computer science classes. (CS classes mein different methods sikhaye jate hain)
* Functional programming paradigm is preferred by many developers. (Functional programming style ko developers pasand karte hain)
* Coding structures are influenced by programming paradigms. (Coding ka structure programming styles se effect hota hai)
